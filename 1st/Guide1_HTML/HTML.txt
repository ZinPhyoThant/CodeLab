Lesson 8 - UL, Li, Image & Path 
<!DOCTYPE html> (Docunent Type ကိုပြောတာ)
<html lang="en"> (language က eng အနေနဲ့သုံးမယ်ပြောတာ)
<head>
    <meta charset="UTF-8"> (Myanmar Font အတွက် သုံးတာ)
    <meta http-equiv="X-UA-Compatible" content="IE=edge"> (Broswer တွေကိုသုံးနိုင်အောင်လုပ်တာ)
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> (Project က responsive website တွေရေးဖို့အတွက်)
    <title>Document</title> (ထိပ်ဆုံးကဟာပဲ)
</head>

<ul>    - unorder list
<ol>    - order list
<li>    -list (သူက ul နဲ့ Li အောက်မှာလိုက်တာ) 
<img>   - src ထဲမှာထည့်တာ
Path    -./ တစ်ခုက တစ်ခါထွက် ../ ဆိုနှစ်ခါထွက်မှာ ../../ ဆိုလေးခါ

Lesson 10 - A tag & iFrame
<a href> a tag က link ချိတ်တဲ့နေရာမှာသုံးတယ် .html# ဆိုပြီး # ကိုယ်ပေါ်စေချင်တဲ့ဟာကိုတန်းရောက်အောင်လုပ်တယ်.
<sup>  က x2 ဆိုရင် 2 ကိုအပေါ်တင်ချင်ရင်သုံး 
<sub> က H2O ဆိုရင် 2 ကိုအောက်ချရင်သုံး
<strike> သူက ရေးထားတာကိုမျဉ်းဖြတ်တာ
<iFrame> သူက အကွက်လေးပေါ်တာ ကြိုက်တဲ့website က ယူူသုံးလို့ရတယ် google map မှာသုံး
